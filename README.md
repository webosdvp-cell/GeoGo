# 🌍 GeoGo

>  An interactive map with routes, labels, gamification, and travel around the world
---

## 📖 English

**GeoGo** is an interactive web map where you can explore the world, create routes, mark favorite places, discover Wikipedia articles, and level up your explorer profile.

### Features
- 🗺️ Multiple map layers (light, dark, satellite, hybrid)
- 📍 Pins & markers with custom labels
- 🚗 Route planning (car, bike, walking) with distance & time
- 📏 Ruler tool — measure distances and area
- 🎲 Random location discovery
- 🎓 Wikipedia articles nearby
- 🎮 Gamification — XP, levels, achievements, daily quests
- 🎵 Background music
- 🔍 Global search
- 💾 Saved routes
- 🎨 Custom profile (avatar, colors, nickname)

### Tech
Leaflet.js • OpenStreetMap • OSRM routing • Wikipedia API • LocalStorage

### How to Use
1. **Search** — type a city name in the top bar
2. **Add pin** — click 📍, then tap anywhere on map
3. **Route** — click 🛣️, select start and end points
4. **Ruler** — click points to measure distance, close for area
5. **Random** — click 🎲 for a surprise location
6. **Profile** — click avatar badge to see achievements & XP
7. **Music** — click 🎵 for relaxing tracks

## 🇷🇺 Русский

**GeoGo** — интерактивная карта для исследования мира, построения маршрутов, создания меток, чтения Википедии и прокачки профиля путешественника.

### Возможности
- 🗺️ Слои карты (обычная, тёмная, спутник, гибрид)
- 📍 Метки с названиями
- 🚗 Маршруты (авто, вело, пешком) с временем и расстоянием
- 📏 Линейка — измерение расстояний и площади
- 🎲 Случайное место
- 🎓 Статьи Википедии поблизости
- 🎮 Геймификация — опыт, уровни, достижения, ежедневные задания
- 🎵 Фоновая музыка
- 🔍 Поиск по всему миру
- 💾 Сохранённые маршруты
- 🎨 Кастомизация профиля (аватар, цвета, ник)

### Технологии
Leaflet.js • OpenStreetMap • OSRM • Wikipedia API • LocalStorage

### Как пользоваться
1. **Поиск** — введите город в верхней панели
2. **Метка** — нажмите 📍, затем кликните по карте
3. **Маршрут** — нажмите 🛣️, выберите начало и конец
4. **Линейка** — кликайте по точкам, замкните для площади
5. **Случайное** — нажмите 🎲, чтобы найти новое место
6. **Профиль** — кликните по аватарке для достижений и опыта
7. **Музыка** — нажмите 🎵 для расслабляющих треков
